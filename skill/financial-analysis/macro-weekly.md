# 宏观周报模块

## 概述

每周生成一份全球宏观经济环境周报，个股分析报告可直接引用而无需重复搜索宏观数据，提高分析效率和一致性。

## 触发方式

```
/financial-analysis 宏观周报
/financial-analysis 本周宏观
```

关键词：`宏观周报`、`本周宏观`、`宏观环境`、`市场总览`

## 周报生成流程

### Phase 1 — 并行数据收集（6路并行）

```yaml
路径A - 全球股市:
  搜索: "{当前年份}年{月}{周} 全球股市周报"
  数据:
    - 美股三大指数（道琼斯/标普500/纳斯达克）周涨跌幅
    - A股（上证/深证/创业板）周涨跌幅
    - 港股恒生指数周涨跌幅
    - 欧洲主要指数
    - 行业板块涨跌排名（前5/后5）

路径B - 货币政策与利率:
  搜索: "美联储 利率 {当前年份}年{月}" + "央行 货币政策"
  数据:
    - 美联储最新表态/会议纪要
    - 美国十年期国债收益率
    - 中国央行公开市场操作
    - 主要央行利率决议

路径C - 经济数据:
  搜索: "{当前年份}年{月} 经济数据 CPI GDP"
  数据:
    - 本周发布的经济指标（CPI/PPI/PMI/非农等）
    - 数据 vs 预期值
    - 关键数据解读

路径D - 地缘政治与贸易:
  搜索: "{当前年份}年{月} 地缘政治 贸易"
  数据:
    - 中美贸易最新动态
    - 地缘冲突进展
    - 关税/制裁变化
    - 重大政策变化

路径E - 大宗商品与外汇:
  搜索: "原油 黄金 价格 {当前年份}年{月}"
  数据:
    - 原油（WTI/布伦特）
    - 黄金/白银
    - 美元指数（DXY）
    - 人民币汇率（在岸/离岸）
    - 加密货币（BTC/ETH）

路径F - 下周展望:
  搜索: "下周 经济日历 {当前年份}年{月}"
  数据:
    - 下周重要经济数据发布日历
    - 下周央行会议/讲话
    - 下周财报季重点公司
    - 下周政策/监管事件
```

### Phase 2 — 分析与定调

```yaml
定调框架:
  美林时钟定位: 当前处于哪个象限（复苏/过热/滞胀/衰退）
  风险偏好: 高/中/低
  资产配置倾向: 
    - 股票（超配/标配/低配）
    - 债券（超配/标配/低配）
    - 大宗商品（超配/标配/低配）
    - 现金（超配/标配/低配）
```

## 个股报告引用

个股分析报告中，在行业与宏观分析章节引用最近的宏观周报：

```markdown
## 宏观环境

> 详见 [[宏观周报-{年}W{周数}]]

本周宏观要点：{从周报提取的1-2个与该标的最相关的要点}
```

引用流程：
1. 检查是否存在本周宏观周报：Glob `/财务分析/宏观周报/宏观周报-{年}W{周数}.md`
2. 若存在：读取并提取相关要点，使用双链引用
3. 若不存在：在个股报告中内联搜索宏观数据（回退到当前行为）
