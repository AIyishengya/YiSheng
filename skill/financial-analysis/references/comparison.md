# 组合对比分析模块

## 概述

支持同行业或同类型标的的横向对比分析，帮助投资者在多个标的间做出更明智的选择。

## 触发方式

```
/financial-analysis 对比 贵州茅台 vs 五粮液
/financial-analysis 对比 CRCL vs COIN vs MSTR
/financial-analysis 对比 宁德时代 vs 比亚迪 vs 亿纬锂能
```

关键词：`对比`、`比较`、`vs`、`PK`

## 对比分析流程

### Phase 1 — 标的识别与分类

```yaml
输入解析:
  - 提取所有标的名称（2-5个）
  - 识别市场类型（A股/美股/港股/混合）
  - 确定行业分类
  - 检查是否有已存在的历史分析报告
```

### Phase 2 — 并行数据收集

```yaml
对每个标的并行搜索:
  - 当前股价与市值
  - 最近一期财务数据（营收、净利、增速）
  - 关键估值指标（PE、PB、PS、PEG）
  - 分析师评级与目标价
  - 近期核心催化剂

行业公共数据:
  - 行业整体增速与趋势
  - 行业平均估值水平
  - 市场集中度（CR3/CR5）
```

### Phase 3 — 多维对比分析

```yaml
对比维度:
  基本面:
    - 营收规模与增速
    - 盈利能力（毛利率、净利率、ROE）
    - 现金流质量
    - 负债水平

  估值:
    - 相对估值（PE/PB/PS 对比）
    - 估值溢价/折价分析
    - PEG 比较（成长性价比）

  市场表现:
    - 近期涨跌幅对比
    - 波动率比较
    - 机构持仓变化

  竞争力:
    - 市场份额
    - 护城河强度
    - 技术/产品领先度

  风险:
    - 各标的风险评分（1-10）
    - 特有风险 vs 共同风险
```
