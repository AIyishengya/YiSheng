# 时政影响分析示例：美联储加息

## 分析过程演示

以下展示使用 `financial-analysis` skill 进行时政事件影响分析的完整流程。

---

### 输入

```
/financial-analysis 分析美联储加息对股市的影响
```

---

### 分析执行过程

#### Phase 1: 事件识别

```yaml
事件信息:
  事件: 美联储加息
  性质: 货币政策/宏观事件
  影响范围: 全球金融市场
  时效性: 实时跟踪

信息需求:
  ✓ 加息幅度与节奏
  ✓ 美联储声明解读
  ✓ 市场预期对比
  ✓ 历史加息周期参考
  ✓ 各类资产影响分析
```

#### Phase 2: 信息收集

```yaml
信源查询:
  1. Reuters/Bloomberg: 最新加息决议
  2. FRED: 历史利率数据
  3. Fed官网: 声明原文
  4. 券商研报: 影响分析

时效性检查:
  | 信息 | 发布时间 | 状态 |
  |------|----------|------|
  | 加息决议 | 2小时前 | ✅ 实时 |
  | 美联储声明 | 2小时前 | ✅ 实时 |
  | 市场反应 | 实时 | ✅ 有效 |
```

#### Phase 3: 影响分析

```yaml
美林时钟定位:
  当前阶段: 经济增长放缓 + 通胀高企
  时钟位置: 滞胀期 → 衰退期过渡
  推荐资产: 现金/债券 → 转向债券

资产影响分析:

1. 债券市场:
   - 短期：收益率上行，价格下跌
   - 长期：收益率曲线倒挂加深

2. 股票市场:
   - 估值压力：PE压缩
   - 板块分化：成长股承压，价值股相对稳健

3. 外汇市场:
   - 美元走强
   - 新兴市场货币承压

4. 商品市场:
   - 黄金：短期承压，长期对冲价值
   - 原油：需求预期下降
```

#### Phase 4: 历史对比

```yaml
历史加息周期参考:

2015-2018年加息周期:
  加息次数: 9次
  幅度: 0.25% → 2.5%
  标普500表现: 上涨约20%
  美元指数: 上涨约15%
  启示: 渐进加息对股市影响有限

2004-2006年加息周期:
  加息次数: 17次
  幅度: 1% → 5.25%
  标普500表现: 上涨约10%
  启示: 快速加息后出现危机

本轮特点:
  - 加息速度历史最快
  - 通胀水平40年最高
  - 需警惕过度紧缩风险
```

### 输出报告

```markdown
# 美联储加息影响分析报告

> 分析时间：2024-03-15
> 事件类型：货币政策

## 事件概述

**决议内容**：美联储宣布加息25个基点，联邦基金利率目标区间上调至5.00%-5.25%

**声明要点**：
- 继续关注通胀风险
- 未来加息路径取决于数据
- 银行业稳定但保持关注

---

## 市场影响

### 直影响

| 资产 | 反应 | 原因 |
|------|------|------|
| 美元指数 | +0.5% | 利差扩大 |
| 美债收益率 | +10bp | 预期兑现 |
| 标普500 | -0.8% | 估值压力 |
| 黄金 | -1.2% | 机会成本上升 |

### 板块影响

| 板块 | 影响方向 | 逻辑 |
|------|----------|------|
| 科技股 | 利空 | 估值对利率敏感 |
| 金融股 | 中性偏多 | 净息差改善 |
| 房地产 | 利空 | 融资成本上升 |
| 必需消费 | 相对抗跌 | 防御属性 |

---

## 受影响标的

### 美股关注

| 标的 | 影响 | 逻辑 |
|------|------|------|
| AAPL | 利空 | 高估值成长股 |
| JPM | 中性 | 净息差与风险平衡 |
| XLF(金融ETF) | 关注 | 利差逻辑 |
| TLT(长债ETF) | 短空长多 | 收益率上行 |

### A股关注

| 标的 | 影响 | 逻辑 |
|------|------|------|
| 贵州茅台 | 间接影响 | 外资流出压力 |
| 宁德时代 | 利空 | 全球成长股估值压制 |
| 银行板块 | 中性 | 国内外息差逻辑不同 |

---

## 投资策略

### 短期策略
- 避免追高高估值成长股
- 关注防御性板块
- 保持一定现金仓位

### 中期配置
- 利率见顶后逐步加仓债券
- 关注估值回调后的优质标的
- 分散配置降低单一风险

---

## 风险提示

1. 加息超预期风险
2. 经济衰退风险
3. 金融系统稳定性风险
```
