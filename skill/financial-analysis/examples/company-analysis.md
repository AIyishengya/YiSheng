# 公司研究示例：特斯拉

## 分析过程演示

以下展示使用 `financial-analysis` skill 进行公司研究的完整流程。

---

### 输入

```
/financial-analysis 研究特斯拉公司的投资价值
```

---

### 分析执行过程

#### Phase 1: 信息收集

```yaml
1. 识别输入类型:
   - 类型: 美股公司
   - 代码: TSLA
   - 名称: Tesla, Inc.

2. 确定信息需求:
   ✓ 公司概况与业务模式
   ✓ 财务数据（SEC EDGAR）
   ✓ 估值与市场预期
   ✓ 竞争格局
   ✓ 管理层分析
   ✓ 最新动态

3. 信息源查询:
   - Yahoo Finance: 股价、财务数据
   - SEC EDGAR: 10-K、10-Q文件
   - Seeking Alpha: 分析观点
   - Reuters: 新闻动态

4. 时效性标注:
   - 股价: 实时数据 ✅
   - 10-K年报: 最新年度 ✅
   - 研报: 近期分析 ✅
```

#### Phase 2: 业务分析

```yaml
业务构成分析:

1. 汽车业务（主要收入来源）:
   - Model 3/Y: 大众市场主力
   - Model S/X: 高端市场
   - Cybertruck: 新产品线

2. 能源业务:
   - 太阳能屋顶/面板
   - Powerwall/Powerpack储能

3. 服务与其他:
   - 超级充电网络
   - FSD自动驾驶订阅

收入结构:
  汽车销售: ~85%
  能源: ~5%
  服务: ~10%
```

#### Phase 3: 框架分析

```yaml
SWOT分析:

Strengths（优势）:
  - 电动车市场领导者
  - 品牌影响力强
  - 技术领先（电池、自动驾驶）
  - 垂直整合能力强

Weaknesses（劣势）:
  - 盈利波动大
  - 产能扩张风险
  - 依赖马斯克个人影响力

Opportunities（机会）:
  - 全球电动车渗透率提升
  - FSD商业化落地
  - 能源业务增长
  - 新市场拓展（如印度）

Threats（威胁）:
  - 传统车企电动化转型
  - 中国车企竞争加剧
  - 政策补贴退坡
  - 自动驾驶监管风险
```

### 输出报告（简化版）

```markdown
# 特斯拉(TSLA) 公司研究报告

> 报告生成时间：2024-03-15

## 摘要

**核心观点**：特斯拉作为全球电动车领导者，具备技术和品牌双重护城河。
但估值处于高位，需关注竞争加剧和盈利能力稳定性。

**投资评级**：持有

---

## 公司概况

| 项目 | 信息 |
|------|------|
| 成立时间 | 2003年 |
| 总部 | 美国德州奥斯汀 |
| CEO | Elon Musk |
| 主营业务 | 电动汽车、能源存储、自动驾驶 |

## 业务分析

### 汽车业务
- 全球电动车市场份额约15%
- 年产能超200万辆
- 主要市场：北美、欧洲、中国

### 能源业务
- 太阳能屋顶部署加速
- Megapack订单饱满

### FSD自动驾驶
- FSD Beta用户超50万
- 自动驾驶出租车计划推进中

---

## 财务概览

| 指标 | 2023 | 2022 | 2021 |
|------|------|------|------|
| 营收(亿美元) | 968 | 815 | 538 |
| 净利润(亿美元) | 150 | 126 | 55 |
| 毛利率 | 18.2% | 25.6% | 25.3% |
| 经营现金流 | 133 | 147 | 115 |

---

## 竞争分析

| 竞争对手 | 优势 | 劣势 |
|----------|------|------|
| 比亚迪 | 垂直整合、成本优势 | 高端品牌力弱 |
| 大众 | 规模效应、渠道 | 电动化转型慢 |
| 蔚来/理想 | 本土化、服务 | 规模较小 |

---

## 风险提示

1. **竞争风险**：中国车企价格战压缩利润空间
2. **政策风险**：IRA补贴规则变化
3. **管理风险**：马斯克精力分散
4. **技术风险**：FSD落地不及预期

---

## 信息来源

1. 财务数据 - [SEC EDGAR](https://www.sec.gov/cgi-bin/browse-edgar?action=getcompany&company=tesla)
2. 股价数据 - [Yahoo Finance](https://finance.yahoo.com/quote/TSLA)
3. 行业数据 - Bloomberg NEF

---

⚠️ 本报告仅供参考，不构成投资建议
```
