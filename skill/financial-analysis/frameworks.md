# 金融分析框架详解

本文档详细说明金融分析中使用的权威评估框架和专家思想。

---

## 一、价值投资框架

### 1. 巴菲特护城河模型 (Moat Analysis)

**核心理念**：寻找具有持久竞争优势的企业，以合理价格买入并长期持有。

**护城河五大来源**：

| 护城河类型 | 定义 | 典型案例 | 评估指标 |
|------------|------|----------|----------|
| **无形资产** | 品牌、专利、牌照 | 茅台、可口可乐 | 品牌溢价率、专利数量 |
| **转换成本** | 更换供应商的成本 | 苹果生态、企业软件 | 客户留存率、流失率 |
| **网络效应** | 用户越多价值越大 | 微信、Facebook | MAU增长、活跃度 |
| **成本优势** | 规模效应或独特资源 | 沃尔玛、煤炭企业 | 毛利率对比、成本结构 |
| **规模优势** | 市场份额领先 | 亚马逊、台积电 | 市占率、产能规模 |

**评估流程**：
```yaml
1. 识别护城河来源
   - 分析主营业务
   - 对比竞争对手
   - 判断竞争优势持续性

2. 量化护城河强度
   - ROE > 15% 连续5年：强护城河
   - ROE 10-15%：中等护城河
   - ROE < 10%：护城河存疑

3. 护城河趋势判断
   - 扩张：竞争优势增强
   - 稳定：竞争优势维持
   - 侵蚀：竞争优势减弱
```

### 2. 格雷厄姆安全边际 (Margin of Safety)

**核心理念**：以低于内在价值的价格买入，为错误留出余地。

**核心指标**：

| 指标 | 格雷厄姆标准 | 说明 |
|------|--------------|------|
| P/E | < 15 | 市盈率不超过15倍 |
| P/B | < 1.5 | 市净率不超过1.5倍 |
| P/E × P/B | < 22.5 | 综合估值指标 |
| 流动比率 | > 2 | 流动性安全 |
| 负债率 | < 50% | 财务稳健 |

**安全边际计算**：
```
安全边际 = (内在价值 - 市场价格) / 内在价值

推荐安全边际：
- 稳定型企业：20-30%
- 周期型企业：40-50%
- 高成长企业：10-20%（需谨慎）
```

### 3. 彼得·林奇 PEG 模型

**核心理念**：将增长率纳入估值考量，寻找被低估的成长股。

**PEG 公式**：
```
PEG = P/E / 盈利增长率 (%)

解读：
- PEG < 1：低估，具有投资价值
- PEG = 1：合理估值
- PEG > 1：高估，需谨慎
- PEG > 2：明显高估，避免买入
```

**林奇投资要点**：

| 类别 | 要点 |
|------|------|
| **业务理解** | 只投资你能理解的公司 |
| **增长类型** | 区分慢增长、稳定增长、快增长、周期、资产隐蔽、困境反转 |
| **规模效应** | 小公司比大公司更容易翻倍 |
| **机构持有** | 机构持股比例低的股票更有潜力 |

## 二、估值模型

### 1. DCF 现金流折现模型 (Discounted Cash Flow)

**适用场景**：现金流稳定、可预测的企业

**计算公式**：
```
企业价值 = Σ [FCF_t / (1 + WACC)^t] + 终端价值 / (1 + WACC)^n

其中：
- FCF：自由现金流 = EBIT × (1 - 税率) + 折旧 - 资本支出 - 营运资本变动
- WACC：加权平均资本成本
- n：预测期数
```

**关键参数设定**：

| 参数 | 设定建议 |
|------|----------|
| 预测期 | 5-10年，根据行业特性 |
| 增长率 | 近期参考历史，远期趋近GDP增速 |
| 折现率 | WACC = 股权成本 × 股权比例 + 债权成本 × 债权比例 |
| 终端增长率 | 通常2-3%（长期通胀率） |

### 2. 相对估值法

**PE 市盈率法**：
```yaml
适用: 盈利稳定的成熟企业
公式: 目标价 = EPS × 目标PE
目标PE确定:
  - 历史分位数：取5年PE的30%-50%分位
  - 行业对比：参考同业平均PE
  - 增长调整：PEG < 1 时PE可适当提高
```

**PB 市净率法**：
```yaml
适用: 金融、地产、制造业等重资产行业
公式: 目标价 = BPS × 目标PB
判断标准:
  - PB < 1：可能被低估或资产质量差
  - PB 1-3：合理区间
  - PB > 3：需验证高ROE支撑
```

**PS 市销率法**：
```yaml
适用: 亏损或微利的高成长企业（如SaaS、互联网）
公式: 目标价 = SPS × 目标PS
参考:
  - SaaS行业：PS通常5-15倍
  - 电商平台：PS通常1-5倍
```

### 3. 资产重置成本法

**适用场景**：重资产行业、困境企业估值

**计算方法**：
```
重置价值 = 土地 + 厂房 + 设备 + 存货 + 无形资产 - 负债

托宾Q值 = 市值 / 重置价值
- Q < 1：市值低于重置成本，可能被低估
- Q > 1：市值高于重置成本，需验证盈利能力
```

## 三、行业分析框架

### 1. 波特五力模型 (Porter's Five Forces)

```yaml
五大竞争力量分析:

1. 现有竞争者竞争强度
   评估指标:
     - 行业集中度(CR4/CR8)
     - 价格战频率
     - 行业增长率
   高竞争特征:
     - 行业分散、增长缓慢、产品同质化

2. 潜在进入者威胁
   评估指标:
     - 进入壁垒（资金、技术、牌照）
     - 规模经济要求
     - 品牌认知度
   高威胁特征:
     - 低壁垒、低资金要求、无明显规模效应

3. 替代品威胁
   评估指标:
     - 替代品性价比
     - 转换成本
     - 消费者倾向
   高威胁特征:
     - 替代品性价比高、转换成本低

4. 供应商议价能力
   评估指标:
     - 供应商集中度
     - 原料重要性
     - 替代供应商可获得性
   高议价能力特征:
     - 供应商集中、原料稀缺、替代困难

5. 买方议价能力
   评估指标:
     - 客户集中度
     - 产品差异化程度
     - 价格敏感度
   高议价能力特征:
     - 客户集中、产品同质、价格敏感
```

### 2. SWOT 分析

```yaml
SWOT 矩阵:

内部因素:
  Strengths (优势):
    - 核心竞争力
    - 财务优势
    - 管理团队
    - 技术专利

  Weaknesses (劣势):
    - 业务短板
    - 财务风险
    - 管理问题
    - 技术落后

外部因素:
  Opportunities (机会):
    - 市场增长
    - 政策利好
    - 技术变革
    - 竞争对手失误

  Threats (威胁):
    - 市场萎缩
    - 政策风险
    - 技术颠覆
    - 新进入者

分析策略:
  SO战略: 利用优势抓住机会
  WO战略: 克服劣势抓住机会
  ST战略: 利用优势规避威胁
  WT战略: 最小化劣势和威胁
```

```yaml
宏观环境分析:

Economic (经济环境):
  - GDP增长率

```

## 四、宏观分析框架

### 1. 达利欧债务周期模型

**核心理念**：经济周期本质上是债务周期

**短期债务周期（5-10年）**：
```yaml
阶段划分:
  1. 扩张期: 信贷增长、资产价格上涨
  2. 泡沫期: 过度借贷、资产价格高估
  3. 衰退期: 去杠杆、资产价格下跌
  4. 复苏期: 债务清理完成、重新扩张

指标监测:
  - 信贷增长率
  - 杠杆率变化
  - 资产价格/收入比
  - 央行政策转向
```

**长期债务周期（50-75年）**：
```yaml
特征:
  - 债务累积超过收入增长
  - 利率接近零下限
  - 央行量化宽松
  - 财富分配失衡

应对策略:
  - 去杠杆（违约/通胀/紧缩）
  - 财富再分配
  - 货币体系改革
```

### 2. 美林时钟模型

**投资时钟**：

| 经济阶段 | 经济增长 | 通胀 | 最佳资产 | 股票行业 |
|----------|----------|------|----------|----------|
| 复苏 | ↑ | ↓ | 股票 | 科技、可选消费 |
| 过热 | ↑ | ↑ | 商品/大宗 | 能源、材料 |
| 滞胀 | ↓ | ↑ | 现金/黄金 | 防御性板块 |
| 衰退 | ↓ | ↓ | 债券 | 公用事业、必需消费 |

**判断指标**：
```yaml
经济增长判断:
  - GDP增速
  - PMI指数
  - 就业数据
  - 工业产出

通胀判断:
  - CPI
  - PPI
  - 核心通胀率
  - 工资增速
```

## 五、风险管理框架

### 1. 风险识别矩阵

| 风险类型 | 具体风险 | 影响程度 | 发生概率 | 应对策略 |
|----------|----------|----------|----------|----------|
| 市场风险 | 系统性风险、波动风险 | 高 | 中 | 分散投资、对冲 |
| 信用风险 | 违约风险、降级风险 | 高 | 低 | 信用分析、限额 |
| 流动性风险 | 无法变现、价差扩大 | 中 | 低 | 保持现金、分批 |
| 经营风险 | 管理失误、战略失败 | 高 | 中 | 深度研究、分散 |
| 政策风险 | 监管变化、政策限制 | 高 | 中 | 政策跟踪、预案 |
| 汇率风险 | 汇率波动 | 中 | 高 | 对冲、本币投资 |

### 2. 仓位管理原则

```yaml
凯利公式:
  最优仓位 = (胜率 × 盈亏比 - 败率) / 盈亏比

保守调整:
  实际仓位 = 最优仓位 × 安全系数(0.25-0.5)

分散原则:
  - 单一标的不超过总仓位10-15%
  - 同一行业不超过总仓位30%
  - 保持5-10%现金应对机会
```

## 六、技术分析框架

### 1. 趋势指标

#### 移动平均线系统 (MA)

```yaml
短期均线:
  MA5/MA10: 短期趋势判断
  金叉（短上穿长）: 买入信号
  死叉（短下穿长）: 卖出信号

中长期均线:
  MA20: 月线，中期趋势
  MA50: 季线，中长期趋势
  MA200: 年线，长期趋势（牛熊分界线）

综合判断:
  多头排列（MA5 > MA10 > MA20 > MA50）: 强势上涨
  空头排列（MA5 < MA10 < MA20 < MA50）: 弱势下跌
  均线粘合后发散: 重要变盘信号
```

#### MACD（指数平滑异同移动平均线）

```yaml
参数: 快线12日, 慢线26日, 信号线9日

信号解读:
  DIF上穿DEA（金叉）:
    - 零轴上方金叉: 强势买入
    - 零轴下方金叉: 弱势反弹，谨慎
  DIF下穿DEA（死叉）:
    - 零轴上方死叉: 回调信号
    - 零轴下方死叉: 强势卖出

背离:
  顶背离: 价格创新高但MACD未创新高 → 见顶信号
  底背离: 价格创新低但MACD未创新低 → 见底信号
```

### 2. 动量指标

#### RSI（相对强弱指标）

```yaml
参数: 通常14日

区间判断:
  RSI > 70: 超买区域，可能回调
  RSI 30-70: 正常区域
  RSI < 30: 超卖区域，可能反弹

进阶用法:
  背离: RSI与价格走势背离是更强信号
  中线突破: RSI突破50为趋势确认
  极端值:
    RSI > 80: 极度超买，高概率回调
    RSI < 20: 极度超卖，高概率反弹
```

#### KDJ（随机指标）

```yaml
参数: K=9, D=3, J=3

信号:
  K上穿D（金叉）: 买入信号
  K下穿D（死叉）: 卖出信号
  J > 100: 超买，可能回调
  J < 0: 超卖，可能反弹

注意: KDJ在震荡市更有效，趋势市容易钝化
```

### 3. 波动指标

#### 布林带 (Bollinger Bands)

```yaml
参数: 中轨=MA20, 上轨=中轨+2σ, 下轨=中轨-2σ

信号:
  价格触及上轨: 超买压力，可能回落
  价格触及下轨: 超卖支撑，可能反弹
  带宽收窄（缩口）: 即将变盘，方向待定
  带宽扩大（张口）: 趋势加速

策略:
  震荡市: 上轨卖出、下轨买入
  趋势市: 沿上轨/下轨运行为强趋势，不宜逆势
```

#### ATR（平均真实波幅）

```yaml
参数: 通常14日

用途:
  波动率衡量: ATR越大波动越剧烈
  止损设置: 止损距离 = 入场价 ± 2×ATR
  仓位管理: 仓位 = 风险金额 / (2×ATR)
```

### 4. 成交量分析

```yaml
量价关系:
  放量上涨: 多头强势，趋势确认
  缩量上涨: 上涨动力不足，警惕回调
  放量下跌: 空头强势，趋势确认
  缩量下跌: 下跌动力衰竭，可能企稳

关键指标:
  换手率:
    < 3%: 冷清
    3-7%: 活跃
    7-15%: 高度活跃
    > 15%: 异常（可能有重大消息）
  
  量比:
    > 2: 明显放量
    1-2: 温和放量
    0.5-1: 缩量
    < 0.5: 明显缩量
```

## 七、量化风险评分系统

### 评分维度与权重

```yaml
总分: 10分（1=极低风险, 10=极高风险）

维度权重:
  财务风险 (25%):
    - 资产负债率 > 70%: +2分
    - 资产负债率 50-70%: +1分
    - 流动比率 < 1: +1分
    - 经营现金流为负: +1分
    - 连续两年净利润下滑: +1分

  估值风险 (20%):
    - PE > 行业均值2倍: +2分
    - PB > 5: +1分
    - PEG > 2: +1分
    - 估值处于历史90%+分位: +1分

  市场风险 (15%):
    - Beta > 1.5: +1分
    - 30日波动率 > 行业2倍: +1分
    - 近30日跌幅 > 20%: +1分

  经营风险 (15%):
    - 单一客户收入占比 > 30%: +1分
    - 单一产品收入占比 > 60%: +1分
    - 管理层频繁变动: +1分
    - 行业处于衰退期: +1分

  政策/监管风险 (10%):
    - 面临重大监管变化: +1分
    - 行业政策收紧: +1分
    - 地缘政治敏感: +1分

  流动性风险 (5%):
    - 日均成交额 < 1000万: +1分
    - 机构持股比例极低: +1分

  特殊事件风险 (10%):
    - 正在被调查/诉讼: +2分
    - 大股东质押比例 > 50%: +1分
    - 商誉占净资产 > 30%: +1分
    - 内部人大量减持: +1分
```
